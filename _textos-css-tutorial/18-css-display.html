CSS Layout - The display Property

O "display: ;" é a propriedade mais importante do CSS para gerenciar layout.

The display Property

O display especifica como um elemento deve ser exibido.

Todos os elementos possuem um valor padrão para essa propriedade, costuma ser inline ou block.

Block-level Elements

Um elemento "block-level (nível de bloco)" sempre começa numa linha nova e toma todo o comprimento disponível.

O elemento div é um elemento block-level.

Veja outros exemplos de elementos block-level:

  • div
  • h1-h6
  • p
  • form
  • header
  • footer
  • section

Inline

Um elemento "inline (na linha)" não começa numa linha nova e ele só terá o comprimento necessário para conter o que há dentro.

O elemento span é um elemento inline.
  • span
  • a
  • img

Display: none;

"display: none;" costuma ser usado com o JavaScript para esconder elementos sem deletar e recriar eles.

O elemento script usa "display:none;" como padrão.

Override the Default Display Value

Você pode mudar o modo de exibição de um elemento. Um exemplo de uso disso é fazer isso no "li" para fazer uma lista horizontal, basta mudar o display para inline.

  • HTML
  • CSS
  • JavaScript

Obs.: especificar o display de um elemento não muda o tipo de elemento, apenas como ele é mostrado. Por isso, se um elemento inline tiver "display: block;" não podemos colocar elementos de bloco dentro dele.

Hide and Element - display:none or visibility:hidden?

Use "display: none;" para esconder um elemento de modo que a página será exibida como se ele não estivesse lá.

"visibility: hidden;" também esconde o elemento, mas ele continuará ocupando o espaço que ocuparia normalmente.

CSS Icons

Para adicionar ícones você pode usar a biblioteca de ícones.

How to Add Icons

Você pode adicionar ícones utilizando um span ou i (ambos entre <>) e o nome do ícone dentro de uma class numa abertura da tag.

Todos os ícones são vetoriais escaláveis (SVG, acho) que podem ser customizados com o CSS (tamanho, cor, somba, etc.

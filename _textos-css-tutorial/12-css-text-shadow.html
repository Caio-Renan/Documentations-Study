CSS Text Shadow

Text Shadow

Use "text-shadow: ;" para adicionar sombra a um texto.

É simples de usar, basta adicionar o valor da sombra horizontal, vertical, efeito de desfoque e a cor.

Você pode adicionar mais de um tipo de sombra no mesmo "text-shadow: ;", basta separar por vírgulas. Aparentemnete ao fazer isso, dependendo de como você configurar, se tiverem cores diferentes, elas vão se misturar.

Veja os exemplos a seguir:

Hello World!!

Hello World!!.

Hello World!!

Hello World!!

Hello World!!

Hello World!!

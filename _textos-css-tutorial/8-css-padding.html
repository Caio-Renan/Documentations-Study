CSS Padding

O padding é usado para modificar a distância entre o conteúdo de uma tag e a sua borda interna.

Eu não tenho padding.

Eu tenho 10px de padding.

Eu tenho 25px de padding.

Você pode adicionar padding para cada lado que desejar, basta colocar "padding-lado: ;", exemplo: padding-top: ;.

Você pode colocar os seguintes valores para o padding:

  • Comprimento - Pode ser em pt, cm, px, in, etc;
  • % - Adiciona um padding em % da largura do elemento que o contém;
  • Inherit - O elemento-filho vai herdar o padding do elemento-pai.

Exemplos:

Eu tenho um padding no topo de 15px, em baixo de 10px, na esquerda de 30px e na direita de 50px.

Eu tenho um padding de 0%.

Eu tenho um padding de 10%.

Eu tenho um padding de 25%.

Eu herdei o padding de meu elemento-pai (15px);

Padding - Shorthand Property

Para definir o comprimento de cada padding coloque os valores no sentido horário (cima, direita, baixo, esquerda) usando o "padding: ;"

Eu tenho um padding no topo de 10px, na direita de 20px, embaixo de de 30px e na esquerda de 40px.

Regra geral:

  • Se tiver 4 valores → cima, direita, baixo, esquerda;
  • Se tiver 3 valores → cima, direita e esquerda, baixo;
  • Se tiver 2 valores → cima e baixo, direita e esquerda;
  • Se tiver 1 valor → todos os lados serão iguais.

Padding and Element width

A propriedade "width: ;" do CSS define o comprimento da área de conteúdo ("box interna, dentro do padding, border, margin") de um elemento.

A quantidade de padding adicionada a um elemento será adicionada ao comprimento total do elemento.

Eu tenho 0px de padding e 300 de width

Eu também tenho 300 de width, mas tenho 25px de padding, logo meu total é de 350px de width.

Para resolver esse problema use a propriedade "box-sizing: ;". Usando "box-sizing" o valor do width irá se manter mesmo que você adicione padding, o que vai mudar é o espaço de conteúdo interno disponível.

Eu tenho 0px de padding e 300 de width

Eu tenho 25px de padding e 300 de width, pois usei "box-sizing: border-box;".

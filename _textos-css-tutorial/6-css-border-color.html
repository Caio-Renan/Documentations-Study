CSS Border Color

Para definir a cor de uma borda use: "border-color: ;".

Você pode selecionar utilizando todos os tipos de definição de cores que você viu até agora.

Se você não selecionar uma cor, ele vai pegar a cor do elemento.

CSS Specific Side Colors

Para definir a cor de carda lado da borda coloque os valores no sentido horário (cima, direita, baixo, esquerda) usando o "border-color: ;".

Texto

Texto

Aparentemente as bordas do lado direito e esquerdo são um trapézio.

Regra geral:

  • Se tiver 4 valores → cima, direita, baixo, esquerda;
  • Se tiver 3 valores → cima, direita e esquerda, baixo;
  • Se tiver 2 valores → cima e baixo, direita e esquerda;
  • Se tiver 1 valor → todos os lados serão iguais.

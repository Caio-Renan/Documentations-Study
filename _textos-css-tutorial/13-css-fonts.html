CSS Fonts

É importante ter uma boa escolha de fonte e de suas propriedades como cor e tamanho.

Generic Font Families and Commonly Used Font Fallbacks

No CSS temos 5 famílias de fontes genéricas:

  • Serif - As fontes terão um traço na borda de cada letra. Cria uma impressão de formalidade e elegância;
  • Sans-serif - As fontes terão somente suas linhas normais. Cria uma impressão moderna e minimalista;
  • Monospace - As fontes terão letras com o mesmo comprimento. Cria uma impressão mecânica;
  • Cursive - As fontes irão se parecer com a escrita humana;
  • Fantasy - As fontes irão ser criativas, "exóticas".

Todas as outras fontes diferentes das que foram citas pertencem a uma dessas fontes genéricas.

Difference Between Serif and Sans-serif Fonts

Minha fonte é da família sans-serif!

Minha fonte é da família serif!

Obs.: Nos computadores, as fontes serif são mais fáceis de ler.

The CSS font-family Property

Para escolher a fonte use "font-family: ;". Comece colocando a fonte que deseja e em seguida coloque uma fonte de família genérica, para o caso do navegador não ter a que você selecionou. Aparentemente essa precaução de ter dar ao navegador opções de fontes para usar é chamada de "fallback".

Obs.: Se o nome da fonte for maior que uma palavra, coloque-a entre aspas simples.

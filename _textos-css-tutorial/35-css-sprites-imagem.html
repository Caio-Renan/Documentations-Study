CSS Image Sprites

Image Sprites

Uma imagem sprite é uma coleção de imagens colocadas em uma única imagem.

Ter várias imagens separadas num site pode torná-lo mais lento, pois será necessário fazer mais requisições, isso sem contar que usará mais largura de banda. Por isso, imagens em sprites podem ser úteis.

Com o CSS nós podemos mostrar somente a parte da imagem que desejamos, para isso primeiro é necessário colocar um fundo transparente como valor do "scr" da tag img, pois ela não pode ficar vazia. Além disso, dê um "id" a tag "img" e use "background: url() ?(left)px ?(top)px;" para adicionar o sprite e definir sua posição. Também defina a altura e o comprimento do sprite que deseja mostrar usando "height" e "width" respectivamente.

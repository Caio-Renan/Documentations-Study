HTML Lists and CSS List Properties

No HTML temos duas listas principais:

  • Lista desordenada - Os itens da lista são marcados com coisas que não possuem ordem (círculos, quadrados, etc.);
  • Lista ordenada - Os itens da lista são marcados com números ou letras que indicam ordem.

O CSS permite:

  • Selecionar marcadores de item da lista diferentes para as listas ordenadas e desordenadas;
  • Definir uma imagem como um marcador de um item da lista;
  • Adicionar uma cor de fundo para as listas e os itens da lista.

Different List Item Markers

Use "list-style-type: ;" para especificar o tipo de marcador.

Alguns valores possíveis são:

  • circle
  • square
  • upper-roman
  • lower-alpha

An Image as The List Item Marker

Use "list-style-image: url();" para especificar uma image como marcador.

Remove Default Settings

Use "list-style-type: none;" para tirar o marcador. Além disso, você também pode tirar o padding e o margin padrão das listas, basta redefinir essas propriedades.

List - Shorthand property

Você pode usar "list-style: ;" para encurtar o código.

Veja exemplos a seguir:

  • list-style: square outside;
  • list-style: circle inside url('../_imagens/li-image.png');

As propriedades devem ser colocadas na ordem a seguir:

  1. list-style-type
  2. list-style-position
  3. list-style-image

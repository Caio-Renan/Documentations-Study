Responsive Table

Uma tabela responsiva vai mostrar uma barra de rolagem horizontal se a tela for muito pequena para mostrar o conteúdo completo.

Use a propriedade "overflow-x: auto;" no div que conter a tabela para fazer com que a tabela fique responsiva.

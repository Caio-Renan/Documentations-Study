CSS Color Values

Você pode selecionar um mesmo tipo de cor de várias formas.

Mesma cor que "tomato":

rgb(255, 99, 71)

#ff6347

hsl(9, 100%, 64%)

Mesma cor que "Tomato", mas com transpârencia de 0.5. Isso faz com que o que estiver atrás dela possa ser visto e a cor em si fica mais transparente. Se você colocar a transparência igual a 0, ela simplesmente some.

rgba(255, 99, 71, 0.5)

hsla(9, 100%, 64%, 0.5)

hsla(9, 100%, 64%, 0)

Todas as cores tem algum tipo de código como essas que estão acima.

CSS Table Alignment

Horizontal Alignmentt

Por padrão o alinhamento de um cabeçalho de uma tabela é centralizado e o alinhamento dos dados são são alinhados a esquerda. Para mudar isso basta usar "text-align: ;".

Veja os exemplos a seguir:

Centralizado
Primeiro nome Último nome
Caio Santos
João Santos

Alinhado a esquerda
Fruta Valor (R$/kg)
Banana 2,50
Maçã 4,20
Melão 3,00

Vertical Alignment

Use "vertical-align: ;" para definir qual será o alinhamento vertical (top, bottom, middle).

Veja os exemplos a seguir:

Dados da lista alinhados ao topo
Primeiro nome Último nome
Caio Santos
João Santos

Dados da lista alinhados ao fundo
Fruta Valor (R$/kg)
Banana 2,50
Maçã 4,20
Melão 3,00

Caption

Use "caption-side: ;" para definit onde a legenda ficará (cima, fundo, etc.).

Observação.: defina o valor de height para perceber o efeito do "vertical-align: ;".

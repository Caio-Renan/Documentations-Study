Horizontal Navigation Bar

Há duas formas de fazer barras de navegação, uma utiliza "float: ;" e a outra usa "display: inline;".

O exemplo acima foi utilizando "float: ;".

Inline List Items

Você pode usar "display: inline;" na tag "li" para fazer com que os itens das lista apareçam um ao lado do outro. Porém, como você já estudou anteriormente, ao colocar esse valor para essa propriedade num elemento, ele não receberá valores como padding, o que é essencial para a criação das barras de navegação. Além disso, ficará um espaço entre os itens que não sai mesmo colocando o margin como 0.

Obs.: a cor do "fundo" da barra de navegação é diferente da dos itens de cada lista intencionalmente. É para ficar mais visível o espaçamento que não gostariamos que existisse.

Active/Current Navigation Link

O exemplo que virá a seguir tem um dos itens da lista selecionado. Para fazer isso você deve adicionar uma classe a uma tag "a" que desejar. Algo interessante é que o efeito de ":hover {}" não irá funcionar nele se você colocar exatamente os mesmos seletores (com adição da classe, é claro) logo após adicionar a propriedae ":hover {}". O mesmo vale para caso queira que ele não tenha a mesma cor que os outros.

Exemplos:

ul a:hover {background-color: #04AA6D; color: white;}

ul a.selected {background-color: green; color: white;}

div.navmenu2 li {background-color: #1898E0;}

div.navmenu2 li.first2 {background-color: #247BA0;}

Você também pode adicionar ":not(.classe-do-elemento-que-não-deve-ser-alterado)" ao hover ou aos seletores que irão alterar alguma propriedade dos elementos. Desse modo, as propriedades que você definir ali não irá afetar aqueles que tem a classe que está entre os parênteses.

div.container2 a:not(.active) {background-color: #333;}

li a.active {background-color: #04AA6D;}

div.container a:hover:not(.active) {background-color: #111;}

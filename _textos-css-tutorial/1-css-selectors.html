CSS Selectors

Temos 5 categorias de seletores:

  • Simple selectors (seleciona elementos baseados em name, id ou class)
  • Combinator selectors (seleciona elementos baseados numa relação específica entre eles)
  • Pseudo-class selectors (seleciona elementos baseados num certo estado)
  • Pseudo-elements selectors (seleciona e estiliza uma parte de um elemento)
  • Attribute selectors (seleciona elementos baseados num atributo ou um valor de atributo)

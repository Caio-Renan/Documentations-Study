CSS Comments

Você pode adicionar um comentário no seu código CSS só para organizar, ele será ignorado pelos browsers.

Para fazer isso basta colocar o que você deseja entre /* */

Você também pode colocar comentários no HTML, basta digitar !-- -->, coloque uma abertura de tag ao lado do ! Eu não sei como fazer o browser entender que eu quero que apareça o que digitei e não que é uma tag.

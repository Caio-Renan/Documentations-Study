CSS background-color

Usando background-color você muda o fundo de uma página.

Exemplo: essa página tem essa cor porque tem isso na stylesheet: "background-color: azure;".

Esse parâmetro pode ser usado para qualquer elemento do HTML, esse aqui é um exemplo.

Que tal adicionar um pouco de transparência? Use "opacity: x;"

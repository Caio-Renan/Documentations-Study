CSS Border - Shorthand Property

Assim como o background, você pode encurtar o código usando "border: ;". Nela você sempre deve especificar o estilo da borda, caso contrário, não irá funcionar.

border: 5px solid steelblue;

Você também pode escolher só um lado, para isso adicione "-lado", exemplo: "border-bottom: ;".

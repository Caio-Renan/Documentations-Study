CSS Text

formatação de texto

Este texto é estilizado com algumas propriedades de formatação de texto. O título usa as propriedades "text-align: ;", "text-tranform: ;" e "color: ;". O parágrafo é identado, alinhado e os espaços entre os caracteres são especificados. O sublinhado foi removido deste "CSS Text" link colorido.

Para fazer esses tipos de estilizações precisamos usar algumas propriedades do CSS.

Text Color

A propriedade "color: ;" pode receber qualquer valor relacionado a cor que você viu até agora (nome da cor, HEX, RGB e HSL).

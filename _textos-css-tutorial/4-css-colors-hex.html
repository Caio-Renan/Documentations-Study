CSS Hex Colors

HEX Value

HEX vem de hexadecimal e tem como padrão: #RRGGBB RR(Red) GG(Green) BB(Blue)

Os valores são entre 00 e ff que é o mesmo que 0 a 255, logo, se você colocar #ff0000 como cor, aparecerá vermelho.

Para mostrar preto coloque todos os valores iguais a 0.

Para mostrar preto coloque todos os valores iguais a ff.

#ff0000

#0000ff

#ffa500

Os tons de cinza também podem ser definidos aqui utilizando o mesmo valor para as 3 variáveis.

#3c3c3c

#616161

#787878

#b4b4b4

#f0f0f0

Em alguns casos você verá esse código ser encurtado para 3 dígitos, isso só ocorre quando os valores estão em pares, tipo: #00ffcc, pode ser escrito como #0fc.

#b58 = #bb5588

#fc9 = #ffcc99

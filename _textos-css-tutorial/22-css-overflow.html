CSS Overflow

Com o "overflow: ;" você irá definir se o conteúdo deve ser cortado para aparecer a barra de rolagem na situação citada acima.

Essa propriedade pode assumir os seguintes valores:

  • visible - É o padrão. O "overflow (talvez seja conteúdo a mais)" não é cortado e o conteúdo restante é renderizado fora da caixa do elemento.
  • hidden - O overflow é cortado e o resto do conteúdo não será visto.
  • scroll - O overflow é cortado e uma barra de rolagem é adicionada para ver o resto do conteúdo.
  • auto - Parecido com o scroll, mas a barra de rolagem só é adicionada quando necessário.

Obs.:

  • Essa propriedade só funciona com elementos de bloco e com altura definida;
  • No OS x Lion (no MAC) as barras de rolagem são escondidas por padrão, mesmo que você coloque "scroll" como a propriedade do overflow.

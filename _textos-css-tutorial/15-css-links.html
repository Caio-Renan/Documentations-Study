Além disso, você pode fazer com que links fiquem com estilos diferentes dependendo do estado em que estão.

Os quatro estados são:

  • a:link - É o estado normal de um link que não foi visitado ainda.
  • a:visited - É o estado de um link que já foi visitado.
  • a:hover - É o estado de um link quando o mouse está em cima dele.
  • a:active - É o estado de um link assim que recebe um click.

Veja os exemplos a seguir:

CSS Links

Observações importantes:

"a:hover" deve vir depois de "a:link" e "a:visited" para ser eficiente.

"a:active" deve vir depois de "a:hover" para ser eficiente.

Se um link tiver como endereço a página que você já está, ele definirá que o link já foi visitado, mesmo você não tendo clicado nele.

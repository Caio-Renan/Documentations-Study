CSS Opacity / Transparency

Use "opacity: ;" para definir a opacidade/transparência de um elemento.

Transparent Image

O valor da opacidade vai de 0 a 1, sendo 1 a opacidade padrão de um elemento e 0 a transparência total (ele não aparecerá).

Veja os exemplos a seguir:

Opacity: 0.2

Opacity: 0.5

Opacity: 1

Transparent Box

Quando você usar "opacity: ;" para adicionar transparência a um fundo, todos os seus elementos-filhos irão herdar essa transparência, o que pode causar alguns problemas.

No exemplo a seguir é possível perceber que fica difícil de ler o texto devido a transparência:

Opacity: 1

Opacity: 0.6

Opacity: 0.3

Opacity: 0.1

Transparency using RGBA

Você pode resolver o problema acima se você colocar transparência ao fundo usando RGBA ou HSLA. Veja como o problema foi resolvido no exemplo a seguir:

Opacity: 1

Opacity: 0.6

Opacity: 0.3

Opacity: 0.1

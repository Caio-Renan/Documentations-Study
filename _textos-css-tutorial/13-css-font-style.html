CSS Font Style

Font Style

Para mudar o estilo de um texto use "font-style: ;".

  • normal - O texto será mostrado normalmente.
  • italic - O texto será mostrado em itálico.
  • oblique - O texto ficará inclinado (muito parecido com o itálico, mas é menos suportado).

Font Variant

Usando "font-variant: ;" você pode dizer se um texto deve ser mostrado em small-caps ou não. Um texto em small-caps tem as suas letras minúsculas transformadas em maiúsculas, mas mantém o tamanho de letras minúsculas.

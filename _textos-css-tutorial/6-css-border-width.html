CSS Border Width

Você pode alterar a largura da borda usando "border-width: ;".

Para isso você pode colocar os valores em in, px, pt, cm, em, etc.

Além dos valores citados anteriormente, você pode colocar alguns já predeterminados, esses são: thin, medium and thick.

Specific Side Widths

Para definir a largura de cada borda coloque os valores no sentido horário (cima, direita, baixo, esquerda) usando o "border-width: ;".

Se você colocar somente dois valores, ele vai entender como o primeiro sendo para o topo e para baixo e o segundo como para a esquerda e para a direita.

Regra geral:

  • Se tiver 4 valores → cima, direita, baixo, esquerda;
  • Se tiver 3 valores → cima, direita e esquerda, baixo;
  • Se tiver 2 valores → cima e baixo, direita e esquerda;
  • Se tiver 1 valor → todos os lados serão iguais.

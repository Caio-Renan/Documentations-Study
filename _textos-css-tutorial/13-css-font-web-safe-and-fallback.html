CSS Web Safe Fonts

What are Web Safe Fonts?

São fontes que estão instaladas em todos os navegadores e aparelhos.

Fallback Fonts

Pode acontecer do navegador ou do aparelho não ter a fonte escolhida por você ou que ela não tenha sido instalada corretamente. Para evitar que isso cause sérios problemas, é ideal colocar outras fontes parecidas com a que você deseja e que provavelmente o navegador terá.

Best Web Safe Fonts for HTML and CSS

A lista a seguir possui as melhores fontes seguras da web:

  • Arial (sans-serif)
  • Verdana (sans-serif)
  • Helvetica (sans-serif)
  • Tahoma (sans-serif)
  • Trebuchet MS (sans-serif)
  • Times New Roman (serif)
  • Georgia (serif)
  • Garamond (serif)
  • Courier New (monospace)
  • Brush Script MT (cursive)

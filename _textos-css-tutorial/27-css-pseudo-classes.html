CSS Pseudo-classes

What are Pseudo-classes?

Uma pseudoclasse é usada para definir um estado especial de um elemento.

Algumas aplicações são:

  • Estilizar um elemento quando o mouse passar por cima dele ou quando receber um click;
  • Fazer com um que o estilo de um link mude após ele já ter sido visitado uma vez;
  • Estilizar um elemento quando ele receber um click.

Syntax

A sintaxe das pseudoclasses é assim:

selector:pseudo-class {property: value;}

A pseudoclasse tem que estar ao lado do ":".

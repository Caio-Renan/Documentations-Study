The CSS Box Model

O modelo de caixa é basicamente a caixa que fica ao redor de cada elemento html.

Ela é formada de várias camadas, a mais externa é o margin, a seguinte é a border, logo depois o padding e por fim o conteúdo.

Para colocar exatamente o comprimento ou altura desejado de um elemento é necessário que você saiba como calcular o comprimento e a altura no modelo de caixa.

O comprimento total deve ser calculado dessa forma:

  • Comprimento total do elemento = width + left padding + right padding + left border + right border + left margin + right margin

A altura total deve ser calculada dessa forma:

  • Altura total do elemento = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

CSS Layout - The z-index Property

The z-index Property

Alguns elementos podem sobrepor outros elementos, para definir qual deverá ficar na frente ou atrás use "z-index: ;".

Um elemento pode ter um "stack order" (refere-se a posição no eixo z) positivo ou negativo. Se um número tem z-index igual a 2 e outro igual a 3, o que tem 3 irá sobrepor o 2, ou seja, aquele que tem o z-index maior ficará por cima dos que tem um menor.

Without z-index

Se você não definir o "z-index: ;" de um elemento, a ordem de sobreposição será baseada na ordem dos elementos no HTML. O último irá sobrepor todos os anteriores, o penúltimo irá sobrepor todos anteriores e assim em diante.

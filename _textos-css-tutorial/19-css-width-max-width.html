CSS Layout - width and max-width

Using width, max-width and margin: auto;

Você pode especificar o comprimento de um elemento usando "width: ;", serve principalmente para os elementos block-level.

Veja os exemplos a seguir:

Eu tenho 45vw de comprimento.
Eu tenho 500px de comprimento.

Use "max-width: ;" para fazer com que o comprimento da div seja variável, ela poderá assumir valores abaixo do valor especificado. Isso faz com que o conteúdo consiga ser visualizado, mesmo com o navegadores estando com um espaço de exibição menor.

Veja os exemplos a seguir (mude o tamanho do navegador para verificar a diferença):

Eu tenho um comprimento máximo de 45vw.

Perceba que o primeiro exemplo age da mesma forma que o último, isso ocorre porque ambos usam vw.

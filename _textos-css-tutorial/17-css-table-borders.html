Full-Width Table

Se você quiser aumentar o comprimento da tabela basta colocar "width: x%;" como propriedade da tabela.

Collapse Table Borders

Perceba que nos casos acima a tabela tem bordas duplicadas. Para resolver isso, use "border-collapse: ;" para definir se quer que as bordas colapsem e virem uma.

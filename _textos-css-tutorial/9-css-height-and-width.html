CSS Height and Width

As propriedades "height: ;" e "width: ;" definem, respectivamente, a altura e o comprimento de um elemento.

Eu tenho 50px de altura e 80% de width

O valor definido na height ou width não leva em consideração padding, margin e borders.

Você pode colocar os seguintes valores para a height e width:

  • auto - É o padrão. O navegador calcula isso.
  • length - Vai definir em px, cm, in, etc.
  • % - Vai definir em porcentagem em relação ao bloco que o contém.
  • initial - Vai fazer os valores voltarem ao seu valor padrão.
  • Inherit - O elemento-filho vai herdar do elemento-pai.

Setting max-width/height and min-width/height

Você também pode definir o máximo e o mínimo de altura e comprimento, basta colocar "min-" ou "max-" antes de definir a propriedade.

Eu tenho 100px de altura e 500px de comprimento máximo. Tente mudar o tamanho da janela para ver o que acontece.

Se você, por acaso, definir um width e um max-width para um elemento e a width ser maior que o max-width, o width será ignorado e o width do elemento será igual ao max-width.

CSS Attribute Selectors

Style HTML Elements With Specific Attributes

É possível estilizar elementos do HTML que tem um atributo ou valores de atributos específicos.

CSS [attribute] Selector

Use o seletor [attribute="value"] para selecionar elementos com um atributo e valor específico.

No exemplo a seguir, todas as tags "a" que tinham "target=''" foram selecionados.

Link 1(não tenho target) Link 2 (tenho "target='_blank'") Link 3 (tenho "target='_self'")

CSS [attribute="value"] Selector

Use o seletor [attribute] para selecionar elementos com um atributo específico.

No exemplo a seguir, todas as tags "a" que tinham "target='_blank'" foram selecionados.

Link 1(não tenho target) Link 2 (tenho "target='_blank'") Link 3 (tenho "target='_self'")

CSS [attribute~="value"] Selector

Use o seletor [attribute~="value"] para selecionar elementos com um atributo e valor específico.

No exemplo a seguir, a cor das bordas variam de acordo com o que está escrito no "title" e o "outline" foi adicionado para as imagens que tinha "kinstone" no "title".

Obs.: Há diferença entre separar com espaço e traços, nesse caso, não separe com traços, pois ele não irá ler, separe com espaços mesmo.

CSS [attribute|="value"] Selector

Use o seletor [attribute|="value"] para selecionar elementos com um atributo começando com um valor específico.

No exemplo a seguir, foi adicionando uma cor de fundo aos elementos que tinham "background" no "title".

Obs.: Há diferença entre separar com espaço e traços, nesse caso, separe com traços, ele vai avaliar o valor que está antes do traço. Se você separar com espaços, ele não vai ler.

Na última imagem está escrito "backgroundred" no "title", mas como a palavra em si não é idêntica ao valor especificado, nada aconteceu.

CSS [attribute^="value"] Selector

Use o seletor [attribute^="value"] para selecionar elementos com um atributo começando com o valor específico.

No exemplo a seguir, foi adicionando uma cor de fundo aos elementos que tinham "background" no "title". Todos começam com background, apesar que o último não há espaço entre "background" e "red".

CSS [attribute$="value"] Selector

Use o seletor [attribute$="value"] para selecionar elementos com um atributo terminando com um valor específico.

No exemplo a seguir, foi adicionando uma cor de fundo diferente aos elementos que tinham "bigger" no "title".

Nesse caso, tanto faz se termina com espaço ou com traços ou se está junto de outra palavra.

CSS [attribute*="value"] Selector

Use o seletor [attribute$="value"] para selecionar elementos com um atributo que contém um valor específico.

No exemplo a seguir, foi adicionando um "outline-offset: ;" aos elementos que tinham "stone" no "title".

Nesse caso, tanto faz se termina com espaço ou com traços ou se está junto de outra palavra.

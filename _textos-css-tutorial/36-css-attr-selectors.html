CSS Attribute Selectors

Style HTML Elements With Specific Attributes

É possível estilizar elementos do HTML que tem um atributo ou valores de atributos específicos.

Abstract

  • [attribute="value"] - Seleciona elementos com um atributo e valor específico;

    Exemplos: [target="_blank"], [type="text"], etc.;

  • [attribute~="value"] - Seleciona elementos com um atributo e palavra específica (separa os nomes por traços irá fazer com que não funcione, separe com espaços);

    Exemplos: [title~="big"], [title~="bold"], etc.;

    O primeiro exemplo afetaria: title="big" e title="big container";

    Não afetariam: title="big-container" e title="bigcontainer";

  • [attribute|="value"] - Seleciona elementos com um atributo começando com um valor específico (se estiver "colado" a outra palavra ou separado por espaços, não funcionará);

    Exemplos: [title|="big"], [title|="bold"], etc.;

    O primeiro exemplo afetaria: title="big" e title="big-container";

    Não afetariam: title="big container" e title="bigcontainer";

  • [attribute^="value"] - Seleciona elementos com um atributo começando com o valor específico (mesmo que esteja junto de outra palavra e separada por espaço ou traço);

    Exemplos: [title^="big"] e [title^="bold"], etc.;

    O primeiro exemplo afetaria: title="big", title="big-container", title="big container" e title="bigcontainer";

  • [attribute$="value"] - Seleciona elementos com um atributo terminando com um valor específico.

    Exemplos: [title$="big"], [title$="bold"], etc.;

    O segundo exemplo afetaria: title="bold", title="text-bold", title="text bold" e title="textbold";

  • [attribute*="value"] - Seleciona elementos com um atributo que contém um valor específico. Ele é bem geral, pode afetar vários elementos.

    Exemplos: [class*=small], [class*=container], etc.;

    O primeiro exemplo afetaria: class="small", class="small-container", class="small container" e class="smallcontainer";

Use o seletor [attribute~="value"] para selecionar elementos com um atributo e palavra específica.

CSS [attribute|="value"] Selector

Use o seletor [attribute|="value"] para selecionar elementos com um atributo começando com um valor específico.

No exemplo a seguir, foi adicionando uma cor de fundo aos elementos que tinham "background" no "title".

Obs.: Há diferença entre separar com espaço e traços, nesse caso, separe com traços, ele vai avaliar o valor que está antes do traço. Se você separar com espaços, ele não vai ler.

Na última imagem está escrito "backgroundred" no "title", mas como a palavra em si não é idêntica ao valor especificado, nada aconteceu.

CSS [attribute^="value"] Selector

Use o seletor [attribute^="value"] para selecionar elementos com um atributo começando com o valor específico.

No exemplo a seguir, foi adicionando uma cor de fundo aos elementos que tinham "background" no "title". Todos começam com background, apesar que o último não há espaço entre "background" e "red".

CSS [attribute$="value"] Selector

Use o seletor [attribute$="value"] para selecionar elementos com um atributo terminando com um valor específico.

No exemplo a seguir, foi adicionando uma cor de fundo diferente aos elementos que tinham "navy" no "title".

Nesse caso, tanto faz se termina com espaço ou com traços ou se está junto de outra palavra.

CSS [attribute*="value"] Selector

Use o seletor [attribute$="value"] para selecionar elementos com um atributo que contém um valor específico.

No exemplo a seguir, foi adicionando um "outline-offset: ;" aos elementos que tinham "stone" no "title".

Nesse caso, tanto faz se termina com espaço ou com traços ou se está junto de outra palavra.

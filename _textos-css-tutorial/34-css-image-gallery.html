CSS Image Gallery

O CSS pode ser usado para criar uma galeria de imagens.

Galeria de imagens são, basicamente, imagens que servem como links. Ao clicar nas imagens, você será direcionado para outra página com aquela imagem ou algum outro conteúdo. Por conta disso, coloque a tag "img" dentro da tag "a".

CSS Font Property

The CSS Font Property

Para encurtar o código das propriedades de uma fonte use "font: ;". Nela você sempre deve especificar o família e o tamanho da fonte, caso contrário não funcionará.

Nela você pode definir todas as propriedades a seguir:

  • font-style;
  • font-variant;
  • font-weight;
  • font-size/line-height;
  • font-family.

Veja os exemplos a seguir:

"font: normal normal normal 25px/35px Arial sans-serif;"

CSS background-attachment

Em alguns casos você pode querer que uma imagem acompanhe
a sua scrollagem ou que ela fique lá onde estava, para mexer nisso você
deve usar "background-attachment".

Quando você coloca "background-attachment: fixed;"
a imagem fica como o Link.

Quando você coloca "background-attachment: scroll;"
a imagem fica como o Chuchu e a chave.

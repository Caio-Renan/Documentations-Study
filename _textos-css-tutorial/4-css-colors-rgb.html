RGB

R(Red) G(Green) B(Blue)

Modelo do parâmetro: rgb(red, green, blue).

Os valores ficam entre 0 e 255. Caso coloque o valor de um deles em 255 e os outros em 0, a cor será igual ao que está com 255.

Se colocar todos iguais a 0, sairá preto.

Se colocar todos iguais a 255, sairá branco.

rgb(255, 0, 0)

rgb(0, 0, 255)

rgb(255, 165, 0)

Tons de escuro podem ser obtidos colocando valores iguais para os 3 parâmetros

rgb(60, 60, 60)

rgb(90, 90, 90)

rgb(210, 210, 210)

rgb(120, 120, 120)

rgb(180, 180, 180)

rgb(240, 240, 240)

RGBA

R(Red) G(Green) B(Blue) A(Alpha)

O canal alpha define o quão transparente a cor vai ser. Você conseguirá ver cada vez mais o fundo caso deixe algo diferente de 1 e deixará totalmente transparente colocando 0.

rgba(255, 99, 71, 0)

rgba(255, 99, 71, 0.2)

rgba(255, 99, 71, 0.4)

rgba(255, 99, 71, 0.6)

rgba(255, 99, 71, 0.8)

rgba(255, 99, 71, 1)

CSS Great Font Pairings

É importante que seus pares de fontes sejam parecidos.

Font Pairing Rules

Veja algumas regras básicas para criar boas duplas de fontes:

  1. Complement

    2. Um bom par de fontes precisam harmonizar bem entre si, não sendo muito diferentes nem iguais entre si.

  2. Use Font Superfamilies

    3. Uma font superfamily são conjuntos de fontes que foram criadas para trabalharem bem juntas.

  3. Contrast is King

    4. O contraste entre as fontes escolhidas é muito importante, já que, se forem muitos similares, haverá conflito. Uma forma de mexer nisso é colocando uma com serif e outra sem serif. Uma boa superfamily costuma ter uma versão serif e sem serif de uma fonte.

  4. Choose Only One Boss

    5. Escolha apenas uma fonte como a "lider". Você pode fazer isso mudando sua cor, tamanho ou peso.

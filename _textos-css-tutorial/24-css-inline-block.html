CSS Layout - display: inline-block

The display: inline-block Value

A principal diferença entre o "display: inline;" e o "display: inline-block;" é que o último te permite definir a altura e o comprimento de um elemento. Além disso o padding-top e o padding-bottom são respeitados, coisa que o "display: inline;" não faz.

A principal diferença do "display: inline-block" para o "display: block;" é que o primeiro não cria uma quebra de linha depois do elemento, permitindo que os próximos elementos fiquem do seu lado, já o segundo o faz.

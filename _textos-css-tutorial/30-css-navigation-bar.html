Navigation Bars

Com o CSS você é capaz de transformar os menus do HTML em barras de navegação.

Navigation Bar = List of Links

Uma barra de navegação é basicamente uma lista de links, por isso use as tags ul, li e a.

Lembre-se de remover os marcadores da tag li usando "list-style-type: none;" e remover o padding e o margin da tag ul.

CSS Font Size

Font Size

Para mudar o tamanho de uma fonte use "font-size: ;". O tamanho padrão caso você não defina é de 16px (1em).

Tamanho absoluto:

  • Define um valor específico para o tamanho da fonte;
  • Não permite que usuários modifiquem o tamanho do texto em qualquer navegador (isso é ruim por razões de acessibilidade;
  • É útil quando sabemos o tamanho físico da saída é conhecido.

Tamanho relativo:

  • Define um valor para o tamanho baseado nos elementos ao redor.
  • Permite que usuários modifiquem o tamanho do texto em qualquer navegador.

Veja a seguri algumas formas de definir o tamanho de uma fonte:

  • px;
  • pt;
  • em;
  • percent;
  • vw.

Veja os exemplos usando px a seguir:

Eu tenho tamanho de 15px.

Eu tenho tamanho de 30px.

Set Font Size With Em

Para permitir o usuário editar o tamanho do texto você pode usar "em" como valor da propriedade "font-size: ;". 1em é igual ao tamanho da fonte atual, como o tamanho padrão nos navegadores é 16px, podemos dizer que, por padrão, em é igual a 16px.

Você pode usar essa fórmula também para converter px em "em": pixel/16 = em.

Eu tenho tamanho de 0.9375em (15px)

Eu tenho tamanho de 1.875em (30px)

Os textos acima tem o mesmo tamanho em px, mas estão em em e, por isso, podem ser editados nos navegadores.

O "em" tem alguns problemas no Internet Explorer, quando você torna ele maior, ele fica maior do que deveria e quando o torna menor, ele fica menor do que deveria.

Use a Combination of Percent and Em

Colocando "font-size: 100%;" em body você resolverá o problema com o Internet Explorer. Todos os navegadores mostrarão o mesmo tamanho de fonte.

Responsive Font Size

Você pode definir o tamanho de um texto usando "vw" que significa "largura da janela de exibição ou viewport width".

Desse modo, o texto vai se manter naquele tamanho mesmo que você mude as dimensões da janela.

Veja a seguir:

Eu tenho tamanho de 1vw.

Eu tenho tamanho de 2vw.

Eu tenho tamanho de 4vw.

Viewport é o tamanho da janela do navegador. 1vw = 1% da largura da janela de exibição. Se a janela de exibição é igual a 100cm, vw será igual a 1cm. Você tem que usar valores pequenos, senão vai ficar horrível.

CSS Margins

As margens são utilizadas para criar espaço entre as bordas externa do elemento e o seu entorno.

Você pode adicionar margin para cada lado que desejar, basta colocar "margin-lado: ;", exemplo: margin-top: ;.

Você pode colocar os seguintes valores para o margins:

  • auto - O browser vai escolher;
  • Comprimento - Pode ser em pt, cm, px, in, etc;
  • % - Adiciona um padding em % da largura do elemento que o contém;
  • Inherit - O elemento-filho vai herdar o padding do elemento-pai.

Valores negativos são permitidos.

Margin - Shorthand Property

Para definir o comprimento de cada margin coloque os valores no sentido horário (cima, direita, baixo, esquerda) usando o "margin: ;"

Eu tenho um margin no topo de 10px, na direita de 15px, embaixo de de 20px e na esquerda de 25px.

Regra geral:

  • Se tiver 4 valores → cima, direita, baixo, esquerda;
  • Se tiver 3 valores → cima, direita e esquerda, baixo;
  • Se tiver 2 valores → cima e baixo, direita e esquerda;
  • Se tiver 1 valor → todos os lados serão iguais.

The auto value

Você pode colocar a margin da direita e da esquerda como auto para fazer com que ele fique horizontalmente centralizado.

The inherit Value

Escolhendo "inherit" como valor de "margin: ;" fará com que a margem desse elemento-filho seja igual a do elemento pai.

Margin Collapse

Se você colocar um "margin-bottom: ;" de um elemento no valor de 50px e um "margin-top: ;" de um elemento logo abaixo desse no valor de 70px, o valor final da distância entre eles não será 120px, mas sim do valor maior entre os dois, que no caso é 70px. Isso vale somente para o bottom e o top.

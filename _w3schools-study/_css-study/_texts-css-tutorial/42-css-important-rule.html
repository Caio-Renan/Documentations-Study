CSS The !important Rule

What is !important?

A regra "!important" no CSS é usada para adicionar mais importância para propriedades/valores que o normal.

Na verdade, se você usar a regra "!important", isso vai substituirá todas as regras de estilizações anteirores para aquela propriedade específica daquele elemento.

Defini que todo "p" terá essa cor e coloquei !important nessa definição.

Defini que o "p" com minha ID terá cor "orangered" e tamanho de 18px.

Defini que o "p" com minha classe terá cor "red" e tamanho de 20px.

Veja que mesmo com a primeira definação tendo menos especificidade, ela substituiu todas as outras que envolviam a cor do elemento "p", mas não as outras propriedades, ou seja, o "!important" funciona para as propriedades que você escolher de um seletor.

Important About !important

A única forma de substituir um regra "!important" é incluir outra regra "!important" numa declaração com mesma (ou maior) especificidade no código fonte, mas aqui começa um problema. Isso faz com que o código CSS fique confuso e a depuração difícil.

Veja o que acontece no exemplo acima, quando todas as propriedades de cor recebem !important.

Veja que no fim ninguém substituiu ninguém.

Maybe One or Two Fair Uses of !important

Uma forma de usar "important!" é se você ter que sobreescrever um estilo que não pode ser sobreescrito de outra forma. Isso pode ocorrer quando você está trabalhando com um Sistema de Gerenciamento de Conteúdo (CMS - Content Management System) e não puder editar o código CSS.

Outra forma é pressupor que você quer um visual especial para botões de uma página.

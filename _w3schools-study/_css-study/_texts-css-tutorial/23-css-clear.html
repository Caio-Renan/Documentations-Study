CSS Layout - clear and clearfix

The clear Property

Ao definirmos um valor para "float: ;" em um elemento, os seguintes irão ser afetados. Para resolver isso, basta utilizar a propriedade "clear: ;".

Essa propriedade pode receber os valores a seguir:

  • none - O elemento não será empurrado para baixo do elemento que está localizado na direita ou esquerda.
  • left - O elemento será empurrado para baixo do elemento que está localizado na esquerda.
  • right - O elemento será empurrado para baixo do elemento que está localizado na direita.
  • both - O elemento será empurrado para baixo do elemento que está localizado na direita ou esquerda..
  • inherit - O elemento herda o valor do clear do elemento-pai.

Você deve espelhar o valor de clear e de float dos elementos que você deseja manipular. Se um elemento está com "float: right;" e você deseja que o seguinte vá para baixo, o valor de clear também deve ser right.

Veja os exemplos a seguir:

Sem usar o clear:

"float: left;".
Sem mudança do valor no clear.

Usando clear:

"float: left;".
"clear: left;".

The clearfix Hack

Se um elemento flutuante for maior que a caixa de conteúdo do elemento, ele irá "vazar". Para resolver isso, basta usar um "clearfix hack".

O primeiro consiste em usar "overflow: auto;" no elemento-pai. Veja os exemplos a seguir:

Sem clearfix:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum, nisi lorem egestas odio, vitae scelerisque enim ligula venenatis dolor...

Com clearflix:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum, nisi lorem egestas odio, vitae scelerisque enim ligula venenatis dolor...

Porém, dependendo do padding e margin dos elementos é possível que apareça uma barra de rolagem. Por isso, muitos usam um clearfix hack mais moderno, veja a seguir:

Sem clearfix:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum, nisi lorem egestas odio, vitae scelerisque enim ligula venenatis dolor...

Com clearflix moderno:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum, nisi lorem egestas odio, vitae scelerisque enim ligula venenatis dolor...

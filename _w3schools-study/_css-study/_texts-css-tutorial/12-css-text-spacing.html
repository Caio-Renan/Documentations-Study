CSS Text Spacing

Text Indentation

Use "text-indent: ;" para colocar indentação na primeira linha do texto.

Veja os exemplos a seguir:

Eu tenho indentação de 10px.

Eu tenho indentação de 30px.

Letter Spacing

Use "letter-spacing: ;" para diminuir ou aumentar a distância entre os caracteres de um texto.

Veja os exemplos a seguir:

Eu tenho um espaçamento entre caracteres de 8px.

Eu tenho um espaçamento entre caracteres de -1px.

Line Height

Use "line-height: ;" para definir a distância entre as linhas de um texto.

Veja os exemplos a seguir:

Eu tenho um espaçamento entre linhas de 25pt. Aut eveniet nihil qui provident inventore qui voluptatem perferendis et praesentium neque. Sed error voluptatum et omnis architecto a illo impedit.

Eu tenho um espaçamento entre linhas de 45px. Aut eveniet nihil qui provident inventore qui voluptatem perferendis et praesentium neque. Sed error voluptatum et omnis architecto a illo impedit.

Word Spacing

Use "word-spacing: ;" para definir a distância entre palavras de um texto.

Veja os exemplos a seguir:

Eu tenho um espaçamento entre palavras de 12pt. Aut eveniet nihil qui provident inventore qui voluptatem perferendis et praesentium neque. Sed error voluptatum et omnis architecto a illo impedit.

Eu tenho um espaçamento entre palavras de -1px. Aut eveniet nihil qui provident inventore qui voluptatem perferendis et praesentium neque. Sed error voluptatum et omnis architecto a illo impedit.

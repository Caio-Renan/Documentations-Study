CSS Text Alignment

Text Alignment

Use "text-align: ;" para definir o alinhamento horizontal do elemento.

Um texto pode ser alinhado a esquerda ou a direita, justificado e centralizado.

Eu estou justificado.

Eu estou centralizado.

Eu estou alinhado a direita.

O justificado faz o texto ter linhas com mesmo comprimento.

Veja o exemplo a seguir:

Aut eveniet nihil qui provident inventore qui voluptatem perferendis et praesentium neque. Sed error voluptatum et omnis architecto a illo impedit. Rem facilis perferendis et aliquam libero et molestiae magni eos fuga magni ea tempora distinctio. Aut aliquam laudantium At quasi officiis est ipsam necessitatibus sit doloribus rerum non ullam excepturi est nesciunt esse.

Text Direction

Se text-direction for da esquerda para a direita, o alinhamento padrão será a esquerda, caso contrário será a direita.

Use "direction: ;" e "unicode-bidi" para a mudar a direção de texto e um elemento.

Veja os exemplos a seguir:

"direction: rtl; unicode-bidi: bidi-override;"

Vertical Alignment

Use "vertical-align: ;" para definir o alinhamento vertical de um elemento.

Veja os exemplos a seguir:

Frase de teste. "vertical-align: text-top";

Frase de teste. "vertical-align: baseline;"

Frase de teste. "vertical-align: text-bottom;"

Frase de teste. "vertical-align: sub;"

Frase de teste. "vertical-align: super;"

Frase de teste. "vertical-align: middle;"

Frase de teste. "vertical-align: bottom;"

Frase de teste. "vertical-align: top;"

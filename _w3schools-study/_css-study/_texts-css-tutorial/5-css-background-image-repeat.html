CSS Background Image Repeat

CSS background-repeat

Por padrão, caso a imagem colocada como fundo não consiga ocupa-lo por completo, ela irá repetir a imagem verticalmente e horizontalmente para que ocupe.







Você pode fazer a imagem ser repetida apenas verticalmente ou apenas horizontalmente usando "background-repeat: x;" para apenas horizontalmente.







Ou usar "background-repeat: y;" para apenas verticalmente.







Você também pode querer não repetir, para isso use "background-repeat: no-repeat;"


Que tal mover essa imagem? Para isso use "background-position: x;"

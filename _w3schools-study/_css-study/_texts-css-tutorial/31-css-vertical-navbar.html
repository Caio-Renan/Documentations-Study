Vertical Navigation Bar

Center Links & Add Borders

Caso queira adicionar bordas para cada item da sua barra de navegação, basta usar "border: ;" na tag "ul" ou na tag "li". Caso queria deixar o texto centralizado, use "text-align: center;" na tag "a".

Full-height Fixed Vertical Navbar

Veja que há uma barra de navegação colada no canto superior esquerdo. Você pode fazer isso usando "position: fixed;", mas não vai funcionar muito bem em aparelhos móveis.

CSS Layout - float and clear

A propriedade "float: ;" especifica como um elemento deve flutuar. Já a propriedade "clear: ;" especifica quais elementos podem flutuar ao lado do elemento "limpo (clear)" e em qual lado.

Veja os exemplos a seguir:

"float: left;"
"float: right;"

The float Property

Use "float: ;" para posicionar e formatar um conteúdo, você pode, por exemplo, colocar uma imagem flutuando na esquerda ou direita de uma caixa.

Essa propriedade pode receber os valores a seguir:

  • left
  • right
  • none
  • inherit

No seu uso mais simples é possível usá-lo para fazer o texto se adequar (wrap pra ser mais exato) a imagem posicionada.

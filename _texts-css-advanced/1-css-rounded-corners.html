CSS Rounded Corners

Use "border-radius: ;" para arredondar os cantos de um elemento.

CSS border-radius Property

A propriedade "border-radius: ;" define o raio dos cantos de um elemento.

Veja os exemplos a seguir:

Cantos arredondados
Cantos arredondados

"border-radius: ;" é uma forma de encurtar o código. Caso deseje, você pode escolher o valor para cada canto, basta usar alguma das propriedades a seguir:

  • border-top-left-radius: ;
  • border-top-right-radius: ;
  • border-bottom-left-radius: ;
  • border-bottom-right-radius: ;

CSS border-radius - Specify Each Corner

Regra geral:

  • Se tiver 4 valores → cima, direita, baixo, esquerda;
  • Se tiver 3 valores → cima, direita e esquerda, baixo;
  • Se tiver 2 valores → cima e baixo, direita e esquerda;
  • Se tiver 1 valor → todos os lados serão iguais.

Veja os exemplos a seguir:

".rounded5 {border-radius: 30px 20px 40px 10px;}"
".rounded6 {border-radius: 30px 20px 40px;}"
".rounded7 {border-radius: 30px 20px;}"
".rounded8 {border-radius: 30px;}"

Você também pode criar bordar elípticas.

Veja os exemplos a seguir:

.rounded9 {border-radius: 60px / 15px;}
.rounded10 {border-radius: 15px / 60px;}
.rounded11 {border-radius: 50%;}

CSS Border Images

With the CSS border-image property, you can set an image to be used as the border around an element.

CSS border-image Property

The property has three parts:

  • The image to use as the border
  • Where to slice the image
  • Define whether the middle sections should be repeated or stretched (esticado)

We will use the following image (called "border.png"):

The border-image property takes the image and slices it into nine sections, like a tic-tac-toe board. It then places the corners at the corners, and the middle sections are repeated or stretched (esticado) as you specify.

Note: For border-image to work, the element also needs the border property set!

Here, the middle sections of the image are repeated to create the border:

An image as a border!

Here is the code:

.border2 { border: 10px solid transparent; padding: 20px; border-image: url(../_images/border.png) 30 round; text-align: center; }

Here, the middle sections of the image are stretched to create the border:

An image as a border!

Here is the code:

.border2 { border: 10px solid transparent; padding: 20px; border-image: url(../_images/border.png) 30 stretch; text-align: center; }

Tip: The border-image property is actually a shorthand property for the border-image-source, border-image-slice, border-image-width, border-image-outset and border-image-repeat properties.

CSS border-image - Different Slice Values

Different slice values completely changes the look of the border:

Example 1:

border-image: url(../_images/border.png) 50 round;

Example 2:

border-image: url(../_images/border.png) 20% round;

Example 3:

border-image: url(../_images/border.png) 50% round;

Example 4:

border-image: url(../_images/border.png) 20 round;

Example 5:

border-image: url(../_images/border.png) 30 round;

Example 6:

border-image: url(../_images/border.png) 40 round;

Example 7:

border-image: url(../_images/border.png) 10 stretch;

Example 8:

border-image: url(../_images/border.png) 20 stretch;

Example 9:

border-image: url(../_images/border.png) 30 stretch;

Example 10:

border-image: url(../_images/border.png) 40 stretch;

Example 11:

border-image: url(../_images/border.png) 50% round;

Example 12:

border-image: url(../_images/border.png) 50% stretch;

Apparently the lines that connecs each corner desappear if you choose a slice value above certain value.

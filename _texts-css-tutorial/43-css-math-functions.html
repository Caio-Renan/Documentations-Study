CSS Math Functions

As função de matemática do CSS permitem expressões matemáticas serem usadas como valores de uma propriedade.

The calc() Function

CSS Syntax

calc(expression)

expression - Necessária. É uma expressão matemática. O resultado será usado como o valor da propriedade. Pode utilizar os operadores: +, -, * e /.

Veja os exemplos a seguir:

Modelo de comparação:

width: 80%;

border: 4px solid salmon;

border-radius: 12px;

Calculando o tamanho de uma div usando uma expressão:

width: cal(80% - 150px);

Calculando o tamanho e o raio de uma borda usando uma expressão:

width: cal(80% - 150px);

border: calc(25px/3) solid salmon;

border-radius: calc(3.33px * 3);

The max() Function

A função max() usa o maior valor, de uma lista devalores separados por vírgula, como o valor da propriedade.

CSS Syntax

max(valor1, valor2, ...)

valor1, valor2, ... - Necessário. Uma lista de valores separados por vírgulas, onde o maior deles será escolhido.

Veja os exemplos a seguir:

Modelo de comparação:

width: 50%;

Calculando o tamanho de uma div usando uma expressão:

width: max(50%, 600px);
width: max(50%, 250px);

The min() Function

A função min() usa o menor valor, de uma lista devalores separados por vírgula, como o valor da propriedade.

CSS Syntax

min(valor1, valor2, ...)

valor1, valor2, ... - Necessário. Uma lista de valores separados por vírgulas, onde o menor deles será escolhido.

Veja os exemplos a seguir:

Modelo de comparação:

width: 50%;

Calculando o tamanho de uma div usando uma expressão:

width: min(50%, 250px);
width: min(50%, 600px);

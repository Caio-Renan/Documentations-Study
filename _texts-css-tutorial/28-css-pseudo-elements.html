CSS Pseudo-elements

What are Pseudo-Elements?

Um pseudo-elemento é usado para estilizar partes específicas de um elemento.

Algumas aplicações são:

  • Estilizar a primeira letra ou linha de um elemento;
  • Inserir conteúdo depois ou antes do conteúdo de um elemento.

Syntax

A sintaxe dos pseudo-elementos é assim:

selector::pseudo-element {property: value;}

The ::first-line Pseudo-element

Use "::first-line {}" para adicionar uma estilização especial para a primeira linha de um texto.

Veja o exemplo a seguir:

Usando "div.container p::first-line" e "div.test p::first-line":

Exemplo

"Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. [...]

Exemplo

Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. [...]

Exemplo

Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum."

Exemplo

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. [...]"

Obs.: Esse pseudo-elemento só funciona em elementos block-level.

Você pode usar as propriedades a seguir nesse pseudo-elemento:

  • font properties
  • color properties
  • background properties
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear

CSS - The ::marker Pseudo-element

Use "::marker {}" para estilizar os marcadores das listas ordenadas ou não ordenadas.

Veja os exemplos a seguir:

Usando "ul.one li::marker" e "ul.two li::marker":

  • Banana;
  • Maçã;
  • Café;
  • Leite.
  • Banana;
  • Maçã;
  • Café;
  • Leite.

CSS - The ::selection Pseudo-element

Use "::selection {}" para mudar a estilização de um texto que tenha sido selecionado pelo usuário.

Clique e arraste para algum lado para ver o que ocorre no exemplo a seguir:

Usando "div.container3 p::selection{}" e "div.container3 h1::selection{}":

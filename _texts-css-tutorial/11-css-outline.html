CSS Outline

Um contorno é uma espécie de "borda" que ficará ao redor da borda, destacando-a. Ela não entra nos cálculos de altura e comprimento, pois fica "fora" do elemento, por isso ela pode acabar sobrepondo elementos próximos.

As propriedades que podem ser definidas para o contorno são:

  • outline-style
  • outline-color
  • outline-width
  • outline-offset
  • outline

CSS Outline Width

A propriedade "outline-width: ;" pode ter os valores a seguir:

  • thin (geralmente 1 px);
  • medium (geralmente 3 px);
  • thick (geralmente 5 px);
  • Uma tamanho específico (in, px, pt, cm, etc.).

CSS Outline Color

A propriedade "outline-color: ;" pode ter como valores todos os tipos de definição de cores que você viu até agora e também tem a opção "invert", que serve para garantir que a cor do contorno seja visível independentemente da cor do fundo.

CSS Outline Style

Para mudar o estilo do contorno use "outline-style: ;".

Ele pode assumir os seguintes valores:

  • dotted - Cria um contorno pontilhado;
  • dashed - Cria um contorno tracejado;
  • solid - Cria um contorno sólido;
  • double - Cria um contorno duplo;
  • groove - Cria um contorno 3D em que o contorno superior esquerdo assumirá um tom diferente da cor selecionada. O efeito depende da cor do contorno. Parece um quadro, da uma impressão de profundidade;
  • ridge - Cria um contorno 3D em que o contorno superior esquerdo assumirá a cor selecionada. O efeito depende da cor do contorno. Parece um quadro, da uma impressão de profundidade;
  • inset - Cria um contorno 3D em que o contorno superior esquerdo assumirá um tom diferente da cor selecionada. O efeito depende da cor do contorno;
  • outset - Cria um contorno 3D em que o contorno superior esquerdo assumirá um tom a cor selecionada. O efeito depende da cor do contorno;
  • none - O elemento fica sem contorno;
  • hidden - Cria um contorno escondido. Não sei bem pra que serve.

Se você não usar "outline-style: ;" ou ao menos selecionar esse parâmetro, as outras formatações de contorno não funcionarão.

CSS Outline Shorthand

Você pode encurtar o código de contorno usando "outline: ;", você deve selecionar alguns dos valores a seguir (a ordem não importa):

  • outline-width;
  • outline-style (necessário);
  • outline-color.

CSS Outline Offset

A propriedade "outline-offset: ;" adiciona espaço entre o contorno e a borda externa. Esse espaço será transparente.

CSS Counters

Contadores CSS são "variáveis" mantidas pelo CSS, que podem receber regras do CSS para monitorar quantas vezes os contadores foram usados.

Os contadores permitem você ajustar a aparência do conteúdo baseado no seu lugar no documento.

Automatic Numbering With Counters

Para trabalharmos com os contadores CSS é preciso usar algumas das propriedades a seguir:

  • counter-reset - Cria ou reseta um contador. Aqui você deve dar o nome a ele, ele será sua "variável";
  • counter-increment - Aumenta o valor do contador. Aqui você deve escolher um contador, ele receberá valores conforme for usado;
  • content - Define o conteúdo que irá aparecer. Você deve escrever o nome do contador onde você desejar que ele apareça;
  • - Adiciona o valor de um contador a um elemento.

Para usar um contador CSS, primeiro é preciso usar um "counter-reset: ;".

No caso acima ficou assim:

Para o h1 - content: counter(first-topic) ". ";

Para o h2 - content: counter(first-topic) "." counter(second-topic) " ";

Para o h3 - content: counter(first-topic) "." counter(second-topic) "." counter(third-topic) " ";

Nos casos acima, foi utilizado "counters" no lugar de "counter" no "content: ;". Usando "counters();", ele mesmo adicionará um novo contador para cada elemento filho que surgir. Você deve escolher o contador mais o que você deseja que venha a seguir (exemplo: counters(contador, ".");, counters(contador, "haha";).

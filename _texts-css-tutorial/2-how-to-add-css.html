How to Add CSS

Nós temos 3 formas de adicionar CSS ao documento, essas são:

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

Serve para quando você já tem um CSS pronto que vai servir para várias páginas, ele ficará fora do documento, num arquivo a parte. Para usá-lo link o seu arquivo HTML ao do CSS colocando dentro da tag: link rel="stylesheet" href="caminho-onde-esta-o-arquivo-css".

O text-indent, o tamanho das letras, a cor de fundo e a font-family foram definidas num arquivo fora daqui. Lá não terão tags, somente seletores e blocos de declaração

Internal CSS

Nesse caso a formatação fica dentro da tag style que fica dentro da tag head. Eu mudei os símbolos das listas dentro desse documento.

Inline CSS

Nesse caso você vai colocar uma formatação para uma tag específica usando o parâmetro style dentro da tag.

Multiple Style Sheets

Em alguns casos você pode acabar formatando um mesmo elemento do seu documento internamente e externamente. A ordem de sobreposição depende de quem você colocou para vir primeiro lá no head, se você linkou o documento ao stylesheet externo antes de fazer seu CSS interno, o que irá se sobrepor é o que veio depois, no caso, o seu CSS interno. Se fizer o contrário, o CSS externo irá sobrepor.

Nesse documento eu coloquei o link primeiro, logo, o tamanho das letras do parágrafo são iguais aos que coloquei no CSS interno.

Cascading Order

Caso tenham muitos estilos para um elemento html, a ordem a seguir será seguida:

  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section)
  3. Browser default

Ou seja, a inline sobrepõe todos, tem o maior nível de prioridade.

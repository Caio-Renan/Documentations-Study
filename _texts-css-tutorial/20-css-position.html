CSS Layout - The position Property

Use "position: ;" para especificar o tipo de método de posicionamento usado para um elemento (estático, relativo, fixado, absoluto ou pegajoso/espesso).

The position Property

A propriedade position pode receber os valores a seguir:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Você pode mexer no posicionamento de elementos usando right, left, top and bottom, mas antes é necessário que você defina o valor de "position: ;" e, dependendo do valor selecionado, essas posições podem se comportar de outras formas.

position: static;

Essa é a posição padrão dos elementos HTML. Nessa posição os elementos não serão afetados por valores como left, right, top ou bottom, ele só irá seguir o fluxo padrão de texto.

Essa div tem a propriedade "position: static;". Além disso, ela tem "right: 50px;". Veja que nada mudou.

position: relative;

Ao usar "position: relative;" você permite que o elemento possa receber valores de right, top, bottom ou top em relação ao elemento que o contém. Além disso, nenhum elemento ocupará o espaço que ele deveria estar ocupando normalmente e o elemento se moverá com base no seu lugar de origem.

Veja os exemplos a seguir:

Elementos que estão centralizados:

Eu não tenho nenhuma propriedade além de "position: relative;"

Eu tenho "right: 3vw;". O elemento foi empurrado para a esquerda em 3vw.

Eu tenho "top: 3vw;". O elemento foi empurrado para baixo em 3vw."

Eu tenho "left: 3vw;". O elemento foi empurrado para a direita em 3vw."

Eu tenho "bottom: 3vw;". O elemento foi empurrado para cima em 3vw."

Eu tenho "right: 3vw", "left: 3vw;", "top: 3vw;" e "bottom: 3vw;".

Elementos que não estão centralizados:

Eu tenho "left: 0;". Repare que o "left: 0;" é o lugar onde ele estaria normalmente.

Eu tenho "right: 10px;".

position: fixed;

Ao usar "position: fixed;" o elemento irá sempre aparecer num espaço baseado na janela de exibição, mesmo que você mexa na barra de rolagem ele irá aparecer.

Ele não deixa um "buraco" no lugar onde deveria estar.

Um exemplo disso é o link que fica a direita da página.

position: absolute;

Ao usar "position: absolute;" o elemento irá ter suas posições baseadas no elemento posicionado que o contém ou que está mais próximo.

Obs.: Elementos com posição absoluta são removidos do fluxo normal de texto e podem sobrepor outros elementos.

Veja os exemplos a seguir:

Eu não tenho nenhuma propriedade além de "position: absolute;"
Eu tenho "right: 0vw;". O elemento foi para o canto direito do "elemento-pai".
Eu tenho "right: 15vw;". O elemento foi empurrado para a esquerda em 15vw.
Eu tenho "top: 0vw;" e "right: 30vw;"". O elemento foi para o topo do "elemento-pai" e andou 30vw da direita para a esquerda.
Eu tenho "top: 15vw;" e "right: 30vw;"". O elemento andou 15vw de cima para baixo e 30vw da direita para a esquerda.
Eu tenho "left: 0vw;" e "bottom: 0vw;". O elemento foi para o canto inferior esquerdo do "elemento-pai".
Eu tenho "bottom: 15vw;". O elemento foi empurrado para cima em 15vw.
Eu estou invadindo o espaço do elemento vizinho.

position: sticky;

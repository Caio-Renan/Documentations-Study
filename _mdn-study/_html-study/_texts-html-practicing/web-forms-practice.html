Abstract and useful informations

  • <li> is used to make the stylization easier;
  • Use 'for' attribute to link the label tag to input tag. For it, put the same name of the input ID. It's good because it create a relation between the label and the form control;
  • The type attribute defines the way the <input> element appears and behaves.
  • Unlike the input tag you will if you want to let something previously typed in a textarea tag, you must write it inside the tag.
  • The main advantage of the <button> element is that the <input> element only allows plain text in its label whereas the <button> element allows full HTML content, allowing more complex, creative button content.
  • You can make a relationship between the label tag and input tag nesting the input tag inside label tag, but it isn't considered the best way of do it.

HTML: HyperText Markup Language

HTML (HyperText Markup Language/Linguagem de marcação de hipertexto) defines the meaning and structure of web content. You can define where some content will stay at and using "tags" you will define if something will be considered as a image, table, paragraph, etc.

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites.

HTML uses "markup (marcações)" to annotate text, images, and other content for display in a Web browser. Some examples of them are: <head>, <title>, <body>, etc.

The name of an element inside a tag is case insensitive. That is, it can be written in uppercase, lowercase, or a mixture.

Introduction to HTML

At its heart, HTML is a fairly simple language made up of elements. Using that you can define if something is a paragraph, a table, a link, etc., and define a document structure, determining if the document has a header, a footer, if has three or two collons, etc.

Getting started with HTML

What is HTML

It isn't a programming language but a markup language.

HTML consists of a series of elements, which you use to enclose, wrap, or mark up different parts of content

Anatomy of an HTML element

The element is the opening tag, followed by content, followed by the closing tag. Take a look in a typical element structure ahead:

Elements and tags aren't the same things. Tags begin or end an element in source code, whereas elements are part of the DOM, the document model for displaying the page in the browser.

Nesting elements

Nesting (aninhamento) is the act of put an element inside of another element. You can't an element inside and outside of another, or it's totally inside or totatlly outside.

Block versus inline elements

  • Block levels elements form a visible block on a page. Use it when you want that your element break to the next rown. A block-level element wouldn't be nested inside an inline element. Block-level elements are usually structural elements on the page like header, paragraph, etc.
  • Inline elements are contained within block-level elements, and surround only small parts of the document's content (not entire paragraphs or groupings of content). It will not cause break row and is often used with text, like links, emphazed parts, etc.

Empty elements

Empty elements are elements used to insert/embed something in document, such as <img>.

Attributes

Boolean attributes

Boolean (boleano) attributes are attributes which has written without values. They can only have one value, which is often equal as the attribute name and is optional. Example: disabled="disabled".

Omitting quotes around attribute values

You can omit an attribute's quotes but it can cause problems, so don't do that.

Single or double quotes?

You can use either one, but after you've choosed one, you only can use the other inside your attribute values.

If you want to use the same quote inside your attribute value, use HTML entities.

HTML comments

Everything inside <!-- --> will be considered as a commentary.

What's in the head? Metadata in HTML

Metadata are data that describe data, for example: informations about who made some document (the document are the "data" and the informations are "metadata"). The head tag is responsible for the metadata of the document.

Adding a title

The <title> element will appear when someone bookmark your page, aside the icon of your page, will be used in search results, etc.

Metadata: the <meta> element

<meta> is the official way of to add metadata to your HTML document.
